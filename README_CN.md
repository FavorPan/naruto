# 🍥 Naruto — 手势追踪忍者之力

基于实时手势追踪的网页应用，通过手势释放**鸣人**和**佐助**的忍术。

## ✨ 功能

| 手势            | 效果        |
| --------------- | ----------- |
| 🖐️ **张开左手** | 鸣人·螺旋丸 |
| 🖐️ **张开右手** | 佐助·千鸟   |

- **实时手势追踪**，基于 [MediaPipe Hands](https://mediapipe.dev/)
- **镜像摄像头**，交互更自然
- **蓝色骨骼叠加层**，确认追踪正常运行
- **Screen 混合模式**，特效动画自然融入摄像头画面
- **渐入渐出**，张开手时威力逐渐增强，握拳时逐渐消散

## 🚀 快速开始

```bash
# 直接用浏览器打开 index.html 即可
open index.html
```

**运行要求：**

- 现代浏览器（Chrome、Edge、Safari、Firefox）
- 摄像头
- 光线充足以获得最佳追踪效果

## 🎮 操作

| 动作                             | 忍术         |
| -------------------------------- | ------------ |
| 张开**左手**（3 根以上手指伸直） | 鸣人·螺旋丸  |
| 张开**右手**（3 根以上手指伸直） | 佐助·千鸟    |
| 握拳                             | 忍术逐渐消散 |

手上的蓝色骨骼线条表示 MediaPipe 正在正常追踪。

## 🛠️ 技术栈

- **MediaPipe Hands** — 手部关键点检测
- **Canvas API** — 骨骼绘制
- **HTML5 Video** — 特效叠加（`mix-blend-mode: screen`）
- **原生 JS** — 零依赖，无需构建
